User-defined wallpaper images for the main menu screen. 

In the "Wallpaper" folder under the game folder, you may set custom wallpaper images 
for different screen modes of the game. 

wp768.jpg - wallpaper file for 1024x768 screen mode 
wp768B.jpg - wallpaper file for 1366x768 screen mode 
wp864.jpg - wallpaper file for 1536x864 screen mode 
wp900.jpg - wallpaper file for 1200x900 screen mode 
wp900A.jpg - wallpaper file for 1440x900 screen mode 
wp900B.jpg - wallpaper file for 1600x900 screen mode 
wp960.jpg - wallpaper file for 1280x960 screen mode 
wp1024.jpg - wallpaper file for 1280x1024 screen mode 
wp1080.jpg - wallpaper file for 1920x1080 screen mode
wp1200.jpg - wallpaper file for 1600x1200 screen mode
wp1440.jpg - wallpaper file for 2560x1440 screen mode
wp2160.jpg - wallpaper file for 3840x2160 screen mode
