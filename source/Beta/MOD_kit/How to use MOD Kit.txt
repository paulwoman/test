The MOD Kit allows you to create a MOD for Capitalism Lab. 

To use the MOD Kit, please unzip the file MOD_kit.ZIP to the current folder \MOD_KIT.

For more details about how to customize a MOD, please see https://www.capitalismlab.com/mod/

If you have any questions about modding, you may post your questions on the official Capitalism Lab forum at http://www.capitalism2.com/forum/
